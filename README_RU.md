Элегантная, многофункциональная и высокопроизводительная нода «всё в одном» для загрузки и обработки изображений в [ComfyUI](https://github.com/comfyanonymous/ComfyUI). Загружайте, кадрируйте, рисуйте маски, настраивайте цвета с помощью профессионального Camera Raw и масштабируйте изображения в едином интерфейсе.

## 🌟 Основные возможности

- **Рабочий процесс «всё в одном»**: Заменяет собой сразу несколько нод для обрезки, цветокоррекции, масштабирования и маскирования.
- **Продвинутый AI-редактор масок**: Выделение объектов кликами (SAM), маскирование по текстовому описанию (GroundingDINO) и быстрое удаление фона нейросетями (RMBG).
- **Профессиональная цветокоррекция**: Полноценный интерфейс Live Camera Raw с ползунками HSL и каналами кривых RGB / Red / Green / Blue.
- **Интерактивный редактор кадрирования, полей и аутпеинта (CPO)**: Симметричное масштабирование, прилипание к границам, пресеты выравнивания, вращение, отражение и выбор цвета заполнения полей.
- **Высокая производительность**: Плавный рендеринг с аппаратным ускорением на GPU и оптимизированным кэшированием холста.

## 🔌 Входы и выходы

- **in_image** *(необязательный)*: Входящее изображение для обработки. При подключении внутренний загрузчик ноды обходится стороной, а порт `original_input` блокируется для избежания логических петель обратной связи.
- **in_mask** *(необязательный)*: Входящая маска для комбинирования с нарисованной.
- **IMAGE**: Финальное обработанное изображение (с кадрированием, цветокоррекцией и масштабированием).
- **MASK**: Сформированная маска (нарисованная вручную или созданная ИИ).
- **original_input**: Оригинальное, необработанное загруженное изображение (активно только при использовании встроенного загрузчика ноды).

## ⚙️ Режимы работы

Интерфейс ноды разделен на 3 переключаемых режима работы.

### 🎬 Режим Base и Live Camera Raw

Используется для быстрой настройки экспозиции, контраста и детальной цветокоррекции. Дважды кликните по вкладке **Base** на ноде, чтобы открыть панель экспресс-настроек.

![Base Mode Node](assets/base1.png)

#### Live Camera Raw:
Нажмите кнопку **Open Live Camera Raw**, чтобы открыть профессиональный интерфейс в стиле Lightroom с расширенными параметрами, HSL и графиком кривых.

| Кривые и базовые ползунки | Панель HSL и детализация |
| :---: | :---: |
| ![Camera Raw Curves](assets/lcr1.png) | ![Camera Raw HSL](assets/lcr2.png) |

#### Особенности и горячие клавиши:
* **Интерактивный рендеринг**: Плавное обновление превью со скоростью 60 FPS на GPU.
* **Пипетка HSL (Иконка с пальцем)**: Зажмите левую кнопку мыши на нужном оттенке изображения и перемещайте курсор влево или вправо для изменения его насыщенности в реальном времени.
* **Инструменты кривых**: Двойной клик на поле графика сбрасывает кривую текущего канала в линейное положение. Клик правой кнопкой мыши по точке удаляет её.
* **Сброс параметров**: Двойной клик по любому ползунку или его текстовому значению возвращает его к значению по умолчанию.
* **Масштабирование и сдвиг**: Прокручивайте колесико мыши для приближения и зажмите среднюю кнопку мыши (колесико) для перемещения по изображению.
* **Выход**: Нажмите клавишу <kbd>Esc</kbd> для закрытия панели без сохранения изменений.

### 🖌️ Режим Mask и продвинутый AI-редактор масок

Режим ручного рисования масок и умного ИИ-выделения. Дважды кликните по вкладке **Mask** или нажмите на иконку **звездочки** на панели ноды для открытия полноэкранного редактора.

![Mask Mode Node](assets/mask1.png)

#### Продвинутый AI-редактор масок:
Сочетает классическую кисть с современными нейросетевыми инструментами сегментации.

| Холст и рисование | Сегментация ИИ (SAM) |
| :---: | :---: |
| ![Canvas & Drawing](assets/mask_e1.png) | ![AI Segmentation](assets/mask_e2.png) |
| **Удаление фона (RMBG)** | **Постобработка и Outpaint-поля** |
| ![Background Removal](assets/mask_e3.png) | ![Post-processing & Outpaint](assets/mask_e4.png) |

#### Инструменты и модели ИИ:
* **Segment Anything (SAM)**: Включите режим SAM, выберите модель и кликайте по объектам для их мгновенного выделения.
  - Поддерживаемые модели: [SAM 2.1 Hiera Tiny](https://huggingface.co/Kijai/sam2-safetensors), [SAM 2.1 Hiera Large](https://huggingface.co/Kijai/sam2-safetensors) и [SAM 3](https://huggingface.co/yolain/sam3-safetensors).
  - Управление кликами: Левый клик — добавить объект (+), Правый клик — исключить область (-).
  - **Режим SAM PRO**: При активации этого режима маска проходит автоматическую постобработку: удаляются мелкие точечные шумы, заполняются пустоты и внутренние отверстия с помощью морфологических фильтров, а также изолируются только те связные области маски, в которых были установлены ваши клики.
* **Маскирование по тексту (GroundingDINO)**: Введите текстовый запрос (например, "glasses", "hair") и нажмите Detect для автоматического создания маски моделью [GroundingDINO SwinT OGC](https://huggingface.co/IDEA-Research/grounding-dino-tiny).
* **Удаление фона (RMBG)**: Мгновенное отделение объекта от фона.
  - Поддерживаемые модели: [InspyreNet](https://huggingface.co/dummy9996/inspyrenet-bf16), [BEN2](https://huggingface.co/PramaLLC/BEN2), [BiRefNet Standard](https://huggingface.co/ezzdev/BiRefNet), [BiRefNet HR](https://huggingface.co/ZhengPeng7/BiRefNet_HR), [BiRefNet Portrait](https://huggingface.co/ZhengPeng7/BiRefNet-portrait) и [BiRefNet Toonout](https://huggingface.co/joelseytre/toonout).
  - **Alpha Matting / Refine Edges (Уточнение краев)**: Активируйте этот чекбокс для точной обработки краев объекта. Модель строит мягкие, полупрозрачные переходы на сложных участках (волосы, шерсть, тонкие нити, прозрачные элементы), что позволяет избежать эффекта «топорного вырезания» и обеспечивает бесшовное наложение.
* **Постобработка маски**: Слайдеры Grow (расширение) и Blur (размытие) маски в реальном времени. Регулировка прозрачности маски на выходе.
* **Pad for Outpaint**: Кнопка быстрого перехода в редактор кропа/полей для увеличения холста с последующим возвратом и сохранением маски.
* **Save to Disk**: Кнопка сохранения маски на диск, оформленная в темно-серый цвет `rgb(42, 42, 42)`.

#### Горячие клавиши:
* <kbd>Alt</kbd> + **Правый клик** + **Драг**: Изменение размера кисти (движение по горизонтали) и ее жесткости (движение по вертикали).
* **Удерживание правой кнопки мыши при рисовании**: Работает как быстрый ластик для стирания.
* <kbd>Ctrl</kbd> + **Левый клик**: Умная заливка замкнутых областей маски.
* <kbd>Shift</kbd> + **Левый клик**: Рисование прямой линии. Зажатие Shift при перемещении фиксирует линии по осям.
* **Колесико мыши**: Zoom холста.
* **Нажатие колесика (СКМ) + Драг**: Перемещение по холсту.
* <kbd>Esc</kbd>: Выход из редактора.

### 📐 Режим Resize и редактор CPO

Для обрезки, добавления полей (Outpaint) и изменения размеров изображения.

| Настройки Resize | Пропорции и заполнение | Выравнивание пикселей |
| :---: | :---: | :---: |
| ![Resize Settings](assets/resize.png) | ![Aspect Lock & Fill](assets/resize2.png) | ![Resolution Snap](assets/resize3.png) |

#### Редактор Crop-Pad-Outpaint (CPO):
Дважды кликните по вкладке **Resize** или нажмите на иконку **Crop** на панели ноды для запуска редактора.

![CPO Editor](assets/cpo1.png)

#### Возможности:
* **Прилипание к границам**: Усилено в 2 раза для облегчения фиксации кроп-рамки на краях исходника.
* **Панель выравнивания**: Кнопки быстрого позиционирования области (Центр, Сверху-Слева, Снизу-Справа и т.д.).
* **Отражение и поворот**: Возможность зеркально развернуть кадр или повернуть на 90 градусов.
* **Цвет для Outpaint**: Палитра из 7 стандартных оттенков + радужное колесо выбора любого цвета заливки.
* **Автомаскирование Outpaint**: При включении, все новые дорисованные поля автоматически преобразуются в маску.

#### Горячие клавиши:
* <kbd>Shift</kbd> + **Перетаскивание угла**: Сохранение пропорций кадра.
* <kbd>Alt</kbd> + **Перетаскивание угла**: Симметричное изменение рамки относительно ее центра.
* **Pixel Snap**: Округление размеров под шаги `x2`, `x4`, `x8`, `x32` или `x64` для совместимости с нейросетевыми генераторами.
* **Колесико / СКМ**: Приближение и перемещение холста.
* <kbd>Esc</kbd>: Закрыть без сохранения.

## ⚠️ Важные замечания

> [!NOTE]
> **Загрузка GroundingDINO**: При первом скачивании текстовой модели DINO может показаться, что процесс завис. Это не так — нода загружает сопутствующие файлы конфигурации и токенизатор с Hugging Face. Дождитесь завершения.

> [!IMPORTANT]
> **Зависание на 100%**: Если индикатор скачивания остановился на 100%, в этот момент идет компиляция библиотек и установка системных зависимостей (особенно для SAM 3). Проверяйте консоль и ждите уведомления об успешной установке в Mask Editor.

> [!TIP]
> **Смещение кисти**: Если после смены входных связей (`in_image` / `in_mask`) вы столкнулись со смещением кисти на ноде, слегка измените размеры ноды на холсте (растяните и верните обратно). Также рекомендуется всегда использовать полноэкранный **Advanced Mask Editor**, где смещения исключены.

> [!WARNING]
> **Ошибки скачивания моделей**: Стабильность загрузки моделей SAM и RMBG зависит от качества сети и доступности Hugging Face. Мы постарались отловить большинство ошибок установки, но не можем гарантировать идеальную работу во всех возможных окружениях. Мы собираем логи падений для исправления багов в обновлениях.

## 🛠️ Установка

### Способ 1: Вручную через Git
1. Перейдите в папку кастомных нод ComfyUI:
   ```bash
   cd ComfyUI/custom_nodes
   ```
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/trx7111/ComfyUI-TrixLoader.git
   ```
3. Перезапустите ComfyUI.

### Способ 2: Через ComfyUI Manager (Install via Git URL)
1. Откройте ComfyUI Manager в браузере.
2. Нажмите кнопку **Install via Git URL**.
3. Вставьте ссылку на репозиторий: `https://github.com/trx7111/ComfyUI-TrixLoader`
4. Нажмите Install и перезапустите ComfyUI.
