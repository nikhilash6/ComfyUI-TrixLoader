## 🧰 Независимый тулбокс и контекстное меню

В версии v2.5 вы можете вызывать все продвинутые редакторы для **любой ноды**, которая производит или принимает изображение. Вы больше не ограничены только нодой Load Image AIO!

![Тулбокс и контекстное меню](assets/toolbox_contextmenu.jpg)

### Особенности и возможности:
- **Плавающий тулбокс (Toolbox)**: Отображается прямо над любой совместимой нодой (например, Load Image, Preview Image, Save Image), предоставляя кнопки быстрого доступа к Camera Raw, Mask Editor и Crop Editor.
- **Правое контекстное меню**: Встраивает функции редактирования в стандартное контекстное меню ComfyUI:
  - *trx adv. camera raw*
  - *trx adv. mask editor*
  - *trx adv. crop/pad/outpaint*
  - *Copy Image / Paste Image / Copy Mask / Paste Mask*
- **Буфер обмена масок**: Позволяет мгновенно скопировать маску с одной ноды и вставить её на любую другую ноду в вашем рабочем процессе.

### 🎬 Live Camera Raw

Профессиональное рабочее пространство в стиле Lightroom с расширенными слайдерами настроек, панелью HSL и цветовыми кривыми.

| Basic Adjustments & pixelize & sketch | HSL Colors & Curves |
| :---: | :---: |
| ![Camera Raw Basic & Curves](assets/cameraraw1.jpg) | ![Camera Raw HSL & Details](assets/cameraraw2.jpg) |

#### Особенности:
- **Плавный интерактивный рендеринг**: requestAnimationFrame-кооперативная отрисовка мгновенно обновляет превью по мере перетаскивания ползунков без задержек.
- **Панель слоев**: Создавайте слои цветокоррекции в боковой панели (режимы наложенного фильтра `filter` или нового изображения `image`).
- **Пресеты пользователя**: Сохраняйте собственные настройки, добавляйте пресеты в избранное, экспортируйте и импортируйте их кодом.
- **Пипетка HSL (Иконка с пальцем)**: Зажмите мышь на оттенке изображения и ведите горизонтально для изменения его насыщенности в реальном времени.
- **Интерфейс кривых**: Точные графики RGB, Red, Green и Blue каналов.

#### Горячие клавиши и фишки:
- **Двойной клик на сетке кривой**: Сбрасывает текущий канал в линейное положение.
- **Клик правой кнопкой по точке кривой**: Удаляет выбранную точку.
- **Двойной клик по слайдеру / значению**: Возвращает параметр к значению по умолчанию.
- **Колесико мыши / Средняя кнопка**: Масштабирование и панорамирование (pan) изображения.
- <kbd>Esc</kbd>: Закрыть редактор без сохранения.

### 🖌️ Продвинутый AI-редактор масок

Умный полноэкранный редактор, сочетающий ручное рисование с нейросетевыми инструментами сегментации.

| ИИ-сегментация SAM | Фокус внимания SAM PRO | Удаление фона RMBG |
| :---: | :---: | :---: |
| ![SAM Segmentation](assets/maskeditor1-sam.jpg) | ![SAM PRO Mode](assets/maskeditor2-sampro.jpg) | ![Background Removal](assets/maskeditor3-rmbg_w_grow.jpg) |

#### Особенности и модели ИИ:
- **Segment Anything (SAM)**: Выделите объект кликами с помощью нейросетей.
  - Поддерживаемые модели: [SAM 2.1 Hiera Tiny](https://huggingface.co/Kijai/sam2-safetensors), [SAM 2.1 Hiera Large](https://huggingface.co/Kijai/sam2-safetensors) и [SAM 3](https://huggingface.co/yolain/sam3-safetensors).
  - **Режим SAM PRO**: Ограничивает рамку обсчета ИИ видимым участком зума (голубой пунктир). Позволяет точнее обходить фон и выделять мелкие детали (волосы).
- **GroundingDINO (Текст-в-Маску)**: Создает маску по текстовому описанию (например, "glasses", "hair") с моделью [GroundingDINO SwinT OGC](https://huggingface.co/IDEA-Research/grounding-dino-tiny).
- **Удаление фона (RMBG)**: Мгновенное вырезание объекта с моделями RMBG ([InspyreNet](https://huggingface.co/dummy9996/inspyrenet-bf16), [BEN2](https://huggingface.co/PramaLLC/BEN2), [BiRefNet](https://huggingface.co/ezzdev/BiRefNet)).
  - **Alpha Matting**: Мягкое сглаживание переходов по краям сложных деталей.
- **Постобработка маски**: Слайдеры Grow (расширение), Blur (размытие) и прозрачности маски.

#### Горячие клавиши и фишки:
- <kbd>Alt</kbd> + **Правый клик** + **Драг**: Регулировка диаметра кисти (горизонтально) и жесткости (вертикально).
- **Правый клик (зажатие и ведение)**: Быстрый переход в режим ластика.
- <kbd>Ctrl</kbd> + **Левый клик**: Умная заливка областей (flood fill).
- <kbd>Shift</kbd> + **Левый клик**: Рисование прямых линий (зажатие Shift выравнивает по осям).
- **Колесико / Средняя кнопка**: Зум и сдвиг холста.
- <kbd>Esc</kbd>: Выход из редактора.

### 📐 Редактор кадрирования CPO

Интерактивный холст для кадрирования, пропорционального масштабирования и увеличения холста (outpaint).

| Rotation, Mirroring & Outpaint Swatches | Interactive Cropping Board |
| :---: | :---: |
| ![Crop Board](assets/cropeditor1.jpg) | ![Outpaint Swatches](assets/cropeditor2.jpg) |

#### Особенности:
- **Симметричный паддинг**: Равномерное добавление или урезание полей со всех сторон.
- **Авто-маска Outpaint**: Новые добавленные поля аутпеинта автоматически записываются в маску.
- **Разрешение с шагом (Snap)**: Округление размеров под кратность (`x2`, `x4`, `x8`, `x32`, `x64`) для полной совместимости с генеративными моделями.
- **Панель выравнивания**: Быстрый сдвиг изображения (по центру, в угол и т.д.).
- **Поворот и зеркалирование**: Отражение по вертикали/горизонтали или поворот на 90 градусов.

#### Горячие клавиши и фишки:
- <kbd>Shift</kbd> + **Перетаскивание угла**: Пропорциональное изменение рамки.
- <kbd>Alt</kbd> + **Перетаскивание угла**: Симметричное изменение рамки от центра.
- **Колесико / Средняя кнопка**: Зум и сдвиг холста кадрирования.
- <kbd>Esc</kbd>: Выход без сохранения.

## 🌊 Нода Load Image AIO и встроенные режимы

Нода **Load Image AIO** объединяет всю подготовку изображений в едином рабочем пространстве. Вместо создания длинных цепочек из множества нод вы можете переключаться между вкладками **Preview**, **Filter**, **Mask** и **Resize** для последовательной настройки изображения в одном удобном интерфейсе.

## 🛠️ Настройки ноды и кастомизация

Вы можете настроить интерфейс и цвета TrixLoader под свой стиль через стандартные **Настройки ComfyUI** (иконка шестеренки):

| Панель настроек | Кастомизация интерфейса |
| :---: | :---: |
| ![Custom style](assets/settings1.jpg) | ![Settings Panel](assets/settings2.jpg) |

- **Potato PC Mode**: Отключает анимации открытия окон редакторов и ограничивает размер превью до `512px` (по умолчанию отключено, превью рендерится в высоком разрешении `1200px`).
- **Цветовая схема ноды AIO**: Выберите цвета для фона, текста, кнопок и активных вкладок Load Image AIO с помощью встроенной палитры.
- **Настройка тулбокса**: Возможность скрыть тулбокс над конкретными нодами, изменить его положение или убрать отдельные инструменты.

## 💾 Пути сохранения и загрузка моделей

- **Путь сохранения**: Промежуточные шаги редактирования сохраняются в папку `ComfyUI/input/aio_input/` (например, файлы `aio_wired_*.png` или `trix_edited_*.png`).
- **Загрузка моделей**:
  - Модели SAM сохраняются по пути `ComfyUI/models/sam2/` или в кэш HuggingFace.
  - RMBG и GroundingDINO сохраняются во внутренний кэш библиотеки HuggingFace.
  - **Примечание по долгой загрузке моделей**: При первом запуске SAM 3 или RMBG загрузка может зависнуть на 100%. Это не баг, а процесс сборки библиотек Python и инициализации Triton-заглушек. Пожалуйста, проверьте консоль терминала и подождите завершения операции.

## ⚠️ Ограничения

1. **Невозможность откатывать кроп и фильтры после сохранения (Save to Node)**: Это обусловлено тем, что мы перезаписываем исходный файл изображения во избежание захламления папки `input` миллионом дубликатов, что стирало бы маску и приводило к черным нередактируемым областям при повторном открытии.
2. **Точность кроп-прилипания (CPO Editor)**: Смарт-снаппинг обрезает пустые паддинг-границы с небольшим округлением, из-за чего изображение может потерять пару крайних пикселей.
3. **Совпадение с Photoshop Camera Raw**: Нам удалось достичь ~95% точности цветокоррекции, однако оригинальные алгоритмы Adobe (кривые, переходы HSL) имеют свои закрытые особенности, которые невозможно скопировать на 100% на открытых технологиях холста.
4. **Выделение SAM 3 по точкам**: Модель может захватить лишние детали. Используйте SAM PRO для ограничения рамки обсчета или переключайтесь в текстовый промпт-режим.

## 🚀 Установка

### Метод 1: Через Git (Вручную)
1. Откройте папку `ComfyUI/custom_nodes/`
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/trx7111/ComfyUI-TrixLoader.git
   ```
3. Перезапустите ComfyUI.

### Метод 2: Через ComfyUI Manager (Поиск по реестру или Git URL)
* **Вариант А (Поиск в реестре)**:
  1. Откройте ComfyUI Manager -> нажмите **Custom Nodes Manager**.
  2. Введите в поиске `trixloader` или `ComfyUI-TrixLoader`.
  3. Нажмите **Install** напротив ноды и перезапустите ComfyUI.
* **Вариант Б (Через Git URL)**:
  1. Откройте ComfyUI Manager -> нажмите **Install via Git URL**.
  2. Вставьте ссылку: `https://github.com/trx7111/ComfyUI-TrixLoader`
  3. Нажмите Install и перезапустите ComfyUI.

## 🎁 Список изменений релиза (v2.5)

### ⭐ Главные изменения
- **Полная независимость от ноды AIO**: Теперь вы можете открывать Camera Raw, Crop Editor и Mask Editor над любой нодой с изображением через плавающую панель тулбокса или контекстное меню.
- **Буфер масок**: Добавлена возможность копирования и вставки маски между нодами.
- **Совместимость масок**: Маски из Adv. Mask Editor полностью совместимы со стандартными масками ComfyUI.

### 🎬 Улучшения в Camera Raw
- **Логическая структура**: Параметры настроек разделены на группы: Basic, HSL, Curves, Details.
- **Пресеты**: Добавлена база пресетов, вкладка избранного, импорт/экспорт с помощью текстового кода.
- **Слои**: Добавлена поддержка создания слоев цветокоррекции в боковой панели (режимы Filter и composite Image).

### ⚙️ Кастомизация и оптимизация
- **Настройка интерфейса**: Интерактивная палитра цветов для элементов ноды AIO, настройка отображения тулбоксов в ComfyUI Settings.
- **Potato PC Mode**: Возможность ограничить превью до `512px` и отключить плавные переходы.
- **Логика выполнения**: Сначала применяются фильтры, а затем кроп/паддинг, что сохраняет цвет добавленных outpaint-полей нетронутым.
- Устранена задержка (lag) при перетаскивании ползунков, оптимизирован рендеринг.
